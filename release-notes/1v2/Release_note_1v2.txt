Title
-------
Heater Remote Control

Description
--------------
The HeaterRC is a simple and easy-to-use application for remotely control your auxiliary heating system or any similar devices which use text messages (SMS) for their controlling.
HeaterRC allows to send a command via text message to the remote system with one single click. It is possible to send commands like start or stop as well as a temperature request command. In addition it is possible to send a timed start request.
The used text for the text messages is fully configurable and can be adapted to your remote system. The default configured text is compatible for vehicle heating systems from Eberspächer, but should be fine also for some other manufacturers (e.g. Webasto). Please read the instruction manual of your remote system and adapt the command configuration if necessary.
The HeaterRC application has a built-in SMS counter which can be seen by clicking on “Menu-> Info”.
---
First step:
All you need to do after the first start-up of the application is to save the cellular phone number of the remote system in the Config-Tab.

TCRiscaldamento

Description
--------------
Il TCRiscaldamento è un applicazione per comandare il vostro sistema di riscaldamento da remoto, facile da utilizzare.
TCRiscaldamento permette di inviare comandi tramite SMS al vostro sistema remoto con un semplice clic. È possibile inviare comandi come “avvio” o “stop” come anche una richiesta di temperatura. In più è possibile inviare un comando per l'avvio temporeggiato.
Il testo dei comandi usati negli SMS è completamente configurabile e puo essere adattato al vostro sistema remoto.Il testo preconfigurato è compatibile con i sistemi di riscaldamento per vetture della Eberspächer, ma dovrebbe andare bene anche con altri manufattori (es. Webasto). Per ulteriori informazioni leggete le istruzioni del vostro sistema remoto e modificate il testo dei comandi se necessario.
L'applicazione TCRiscaldamento possiede un contatore integrato per gli SMS inviati che puo essere chiamato cliccando su “Menu->Info”.
---
Primo passo:
Al primo avvio del applicazione è necessario inserire e salvare il numero cellulare del vostro sistema remoto che volete telecomandare.
---
Ogni tipo di feedback è il benvenuto e verra considerato se possibile nei prossimi aggiornamenti.

Standheizung Fernbedienung

Description
--------------
Die StandheizungFB ist ein simples und einfach zu bedienendes App für fernsteuerbare Standheizungen oder ähnliche Geräte welche mittels SMS Nachrichten gesteuert werden.
StandheizungFB erlaubt das senden eines Befehls mittels SMS durch einen einzigen Knopfdruck. Man kann Befehle wie “Start” oder “Stop” sowie auch eine Temperaturanfrage senden. Zusätzlich kann man auch einen zeitgesteuerten Startbefehl schicken.
Der Befehlstext ist editierbar und kann an Ihr ferngesteuertes System angepasst werden. Der vorkonfigurierte Befehlstext ist kompatible mit Standheizungssystemen von Eberspächer, sollte aber auch mit anderen Herstellern funktionieren (z.B. Webasto). Bitte lesen Sie die Anleitung ihres ferngesteuerten Systems und bearbeiten Sie den Befehlstext falls notwendig.
StandheizungFB hat einen integrierten SMS-Zähler welcher unter „Menu->Info“ abgerufen werden kann.
---
Erster Schritt:
Beim ersten Start der App, muss die Mobiltelefonnummer des ferngesteuerten Systems im Konfig-Tab eingetragen werden.
---
Feedback zu dieser App ist herzlich Willkommen und wird falls möglich im nächsten Update berücksichtigt.

TCChauffage

Description
--------------
Le TCChauffage est une application facile a utiliser pour télécommander vôtre chauffage autonome ou appareil similaire que utilisant message SMS pur les commandes.
TCChauffage vous offre la possibilité de commander votre chauffage avec un seule pression du bouton.
C'est possible envoie commande comme «Démarrer», «Arrête» et aussi la demande de température. Additionnelle c'est possible l'envoi du commande de démarrage programme.

Le texte de commande est modifiable et vous le pouvez adapter a votre dispositif télécommande. Les texte qui est déjà configurée est compatible avec les dispositif de chauffage de Eberspächer, mais peux fonctionner aussi avec autre système (p.ex. Webasto). S.v.p. lire le mode d'emploi de votre dispositif télécommande et si nécessaire modifiez le texte des commandes. 
TCChauffage offre un compteur intégré pour les SMS qui est visible avec «Menu->Info». 

---
Premier pas:
A la premier démarché de cette application est nécessaire l'insertion du numéro de votre dispositif télécommande dans l'onglet 'config'.

CRcalorífero

Description
--------------
El CRcalorífero es una aplicación para el control de su sistema de calefacción a distancia, fácil de usar.
CRcalorífero le permite enviar comandos a través de SMS a su sistema control remoto con un solo clic. 
Puede enviar comandos como "Encender" o "Apagar", así como una solicitar de temperatura. Además, puede enviar un comando para iniciar programado.
El texto de los comandos utilizados en SMS es totalmente configurable y se puede adaptar a su sistema remoto. 
El texto preconfigurado es compatible con los sistemas de calefacción de los coches de Eberspächer, pero debe ir bien con otros fabricantes (por ejemplo, Webasto). 
Para obtener más información, lea las instrucciones de su sistema de control remoto y cambiar el texto de los controles si es necesario.
El CRcalorífero aplicación tiene un contador integrado de SMS enviados que se puede llamar haciendo clic en "Menu->Info".
---
Paso uno:
La primera vez que inicie la aplicación que deben ingresar y guardar el número de teléfono del sistema remoto que desea utilizar.
